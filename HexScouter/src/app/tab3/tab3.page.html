Tutorial
  1. Set Name and Scouting Position
    1. Navigate to the Settings Tab
    2. Input name and Scouting Position as well as which regional you are scouting at
    3. The app will automatically save your settings when you navigate to another screen
  2. Start filling out the Scouting Form under "New Form"
  3. Forms lets you manage your finished forms
    • You can now scout offline!
    • Forms are saved to your device after you are finished
    • You can view these forms, edit them, and delete them
    • When you're back online, click the "Upload" button to save all your entries to the google sheet
    • Note: This will only work if you have set which regional you are scouting at
  4. The "View" Tab lets you view the current google sheet.
    • This will only work if you have set which regional you are scouting at
